Finance KPI Dashboard

Monitoring Financial Metrics and Sales Performance

Project Overview

I developed a dynamic Finance KPI Dashboard using Power BI to track key financial metrics and sales performance. The dashboard delivers real-time insights, empowering data-driven decision-making for businesses.

Data Preparation & Modeling

Data was transformed using Power Query, where I cleaned and organized the dataset. A data model was built to establish relationships between tables, and DAX was used for key calculations like total sales and variance analysis.

Key Features

  • Interactive filters for team-specific insights
  • Real-time sales tracking against targets
  • Performance analysis with variance tracking

Technologies Used

  • Power BI for data visualization and reporting
  • Power Query for data cleaning and transformation
  • DAX for creating custom metrics and KPIs

Spotify Music Analysis

Analyzing Music Evolution

Project Overview

This project explores how music has evolved over time, analyzing Spotify track data across 125 genres. Using Python for data processing, I identified trends in song characteristics and genre popularity, offering insights into changing musical preferences.

Motivation

As an avid Spotify listener and a student of music history, I have always been fascinated by how music evolves over time. This project was inspired by my interest in exploring the dynamic shifts in musical trends and user behaviors. Analyzing these changes offers a unique perspective on the cultural and technological factors influencing the music industry

Approach

The analysis was conducted using Python with libraries such as Pandas and NumPy for data manipulation. Visualization was achieved using Matplotlib and Seaborn. The focus was on examining track metadata to identify trends in song duration and genre popularity

Key Visualizations

Here are some insights from the project visualizations:

  • Correlation Between Loudness and Energy
  • Trends in Song Duration Over Time
  • Genre Popularity by Year

Results

The analysis revealed that shorter, high-energy songs have become increasingly popular, particularly in modern genres like pop and hip-hop. These insights help explain current trends in music production and consumption.

Technologies Used

  • Pandas, NumPy for data processing
  • Matplotlib, Seaborn for visualization
  • Jupyter Notebook for analysis

Hypothyroidism Analysis

Understanding Thyroid Disorders

Project Overview

This project analyzes hypothyroidism cases to uncover patterns and key factors influencing the condition. By leveraging Python and SQL, the study provides actionable insights for better diagnosis and management.

Motivation

As someone diagnosed with hypothyroidism at a young age, I’ve always been curious about the factors contributing to this condition. This project explores those patterns to provide insights that could help others in similar situations.

Objectives

  • Perform exploratory data analysis (EDA) to understand the dataset’s distribution and insights.
  • Identify the most affected demographics (age, gender) for hypothyroidism.
  • Analyze key lab results to aid in early diagnosis.

Dataset

  • Source: Kaggle - Thyroid Disease Detection
  • Classes: Hypothyroid and Non-Hypothyroid
  • Features: Age, gender, medical history, thyroid test results
